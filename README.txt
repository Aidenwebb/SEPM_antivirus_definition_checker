# Symantec Endpoint Encryption Manager Definition checks

## Installation

Just run the file

## Usage

Config file Key:
  "database_server": - Host name of your database server
  "database_port": - Port of your database service
  "database_user": - Username of user with read access to database
  "database_pass": - Database users password
  "database_name": - Name of the SEPM database
  "execution_log_file": - Runtime information and exception log location
  "report_file": - Report csv for definitons older than threshold
  "report_threshold_older_than_days": - Threshold of when definitions are considered outdated
